Funcionalidades

Tudo que você precisa
num só app

Do Windows 11 ao Arch Linux — detecção automática de tipo de ISO e configuração sem complicação.

⊞

Windows ISO Perfeito

Detecta automaticamente ISOs Windows via estrutura ISO 9660. Configura GPT + UEFI + NTFS e faz o split de install.wim maior que 4 GB em C puro — sem wimtools obrigatório.

🐧

Linux ISO Raw (dd)

Cópia raw preservando o boot híbrido BIOS+UEFI embutido na ISO. Suporte a Ubuntu, Fedora, Arch, Debian e qualquer distribuição compatível.

🔍

Detecção Inteligente

Lê a estrutura real ISO 9660 — não apenas a extensão do arquivo. Ao selecionar a ISO, auto-configura partição, filesystem e modo de boot.

🌐

Bilíngue PT-BR / EN

Toda a interface se traduz com um clique no headerbar. Sem necessidade de reiniciar o app ou alterar configurações do sistema.

🔒

Sem Terminal

Autenticação gráfica via polkit — a janela de senha do sistema aparece automaticamente. Instale pelo .deb, AppImage ou .rpm e abra pelo menu de aplicativos.

⚡

Assembly x86-64

6 funções críticas implementadas em Assembly puro com NASM: cópia otimizada REP MOVSB, checksum, detecção de extensão e mais.

Instalação

Instale em segundos

Escolha o formato para sua distribuição. Todos pedem senha graficamente — sem nenhum terminal.

bash 
# Baixe o .deb na página de Releases
wget https://github.com/ailtonmartins/rutlix/releases/latest/download/rutlix_2.1_amd64.deb

# Instale com dpkg
sudo dpkg -i rutlix_2.1_amd64.deb
sudo apt-get install -f   # corrige dependências se necessário

# Pronto! Procure "Rutlix" no menu de aplicativos

Internamente

Assembly x86-64 puro

6 funções críticas escritas em NASM — sem libc, direto nas syscalls. Fallback automático para C puro se NASM não estiver disponível.

rutlix/asm/utils.asm
Função Instrução / Syscall Descrição
asm_get_file_size sys_stat Tamanho da ISO sem libc
asm_str_ends_with_iso OR 0x20 bitmask Verifica extensão .iso
asm_compute_checksum Loop unrolled 8× Validação de dados
asm_memcpy_fast REP MOVSB Cópia otimizada pelo microcode
asm_zero_memory REP STOSB Zera buffers
asm_align_up Aritmética bitmask Alinhamento de blocos
